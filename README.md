🚀 Smart Study Sidekick — AI-Powered Productivity & Learning Platform
(Full-Stack Project | React + TypeScript + Supabase + Llama 3.3 70B)

📌 About the Project
Smart Study Sidekick is a full-stack AI productivity platform designed to help students, developers, and professionals manage their daily workflow more efficiently. It combines AI assistance, task management, finance tracking, goal planning, project organization, and study workflow automation in one centralized dashboard.
The platform provides an intelligent workspace powered by the Llama 3.3 70B model through Supabase Edge Functions, delivering real-time AI responses for learning, debugging, resume optimization, and more. Every feature is fully integrated into a seamless UI built with modern component architecture and a responsive dashboard environment.

🛠️ Tech Stack Used
Frontend
-React + TypeScript + Vite
-TailwindCSS for utility-first styling
-shadcn/ui + Radix UI for a production-grade component system
-Recharts for data visualization
-Sonner & Custom Toast System for notifications
-React Query for cached server state & API management

Backend
Supabase for:
-Authentication
-Database
-Realtime functionality
-Storage
-AI & Edge Computing
-Supabase Edge Functions to run AI logic

Llama 3.3 70B Model for:
-Chatbot responses
-Code debugging
-Resume analysis / ATS scoring

Architecture
-Component-driven UI with 60+ reusable components
-Modular folder structure
-Secure API calls with RLS rules
-Resizable panels, command palette, sheet drawers, sidebar system
-Fully responsive, mobile-friendly layout

🎯 Key Features
-AI Chatbot with streaming responses
-Code Fixer (AI debugging engine)
-Resume ATS Analyzer with skill gap detection
-Project Manager
-Finance Tracker with charts
-Goal Progress Tracker
-Daily Planner & Task Scheduler
-Pomodoro Study Timer with session logging
-Resource Hub for curated study links

🌍 Real-World Use & Benefits
Smart Study Sidekick solves practical problems for students and working professionals:
-Helps organize daily tasks and stay on track
-Provides AI-powered learning help for coding, problem-solving, writing, and revisions
-Saves time by automating resume optimization, code debugging, and task planning
-Improves personal productivity through goal tracking, finance monitoring, and structured study cycles
-Acts as a centralized workspace—users no longer need 5–6 different apps for learning and planning
-This project demonstrates how AI and modern full-stack engineering can enhance everyday personal productivity.
